## 摘要
多模态智能体检索增强生成（RAG）系统将攻击面从提示注入扩展到文本毒化、图像注入、直接查询攻击和编排器级别的工具操纵。现有的对抗测试方法通常是特定于表面的，并且经常重复使用已知的攻击模板；在文本毒化基准测试中，我们测量到73-84%的完全重复。我们提出了MIRROR，一个统一的跨表面框架，执行记忆引导的蒙特卡洛树搜索，同时在明确的新颖性约束下，根据检索到的上下文调节候选生成。确定性新颖性门拒绝任何与检索集匹配的候选（在归一化比较下），从而允许检索信息用于搜索先验，而不启用提示复制。在多模态智能体RAG目标的四个攻击面上，MIRROR在图像毒化上达到了76%的攻击成功率，而基线为52%；在编排器攻击上以一半的查询成本达到了97%的攻击成功率，并且具有最低的跨表面方差（变异系数0.47）。相比之下，专门的基线在跨表面上崩溃：后缀优化在文本毒化上达到79%的攻击成功率，但在直接查询上仅为1%。我们发布了ART-SafeBench，包含41,815条包内记录和运行时适配器，在四个表面上总共产生41,991+条记录。

## Abstract
Multimodal agentic retrieval-augmented generation (RAG) systems expand the attack surface beyond prompt injection to include text poisoning, image injection, direct-query attacks, and orchestrator-level tool manipulation. Existing red-teaming approaches are typically surface-specific and often recycle known attack templates; on text-poisoning benchmarks we measure 73-84% exact duplication. We present MIRROR, a unified cross-surface framework that performs memory-guided Monte Carlo tree search while conditioning candidate generation on retrieved context under an explicit novelty constraint. A deterministic Novelty Gate rejects any candidate matching the retrieval set under normalized comparison, allowing retrieval to inform search priors without enabling prompt copying. Across four attack surfaces on a multimodal agentic RAG target, MIRROR attains 76% ASR on image poisoning compared with 52% for baselines, 97% ASR on orchestrator attacks at half the query cost, and the lowest cross-surface variance (coefficient of variation 0.47). In contrast, specialized baselines collapse across surfaces: suffix optimization reaches 79% ASR on text poisoning but 1% on direct queries. We release ART-SafeBench with 41,815 in-package records and runtime adapters yielding 41,991+ total records across four surfaces.

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：MIRROR采用**双阶段架构**：① **记忆银行**存储历史成功攻击trace（含元数据），② **新颖性约束下的MCTS规划**。通过检索记忆提供操作先验和拒绝集，同时使用确定性**新颖性门**（Novelty Gate）防止候选与基准池或当前会话中已接受的提示重复。

- **关键技术细节**：
  - **记忆表示与检索**：每个记忆包含攻击表面、目标、提示及执行trace。Prior Network维护持久化向量索引（余弦相似度），检索k-NN记忆提供操作先验和拒绝集。
  - **新颖性门**：定义归一化函数`norm_ex`（空白正则化）和`norm_alnum`（小写字母数字）。候选x通过两个归一化后均不与基准池或会话内已接受集匹配，才允许进入后续搜索。这提供了确定性、可审计的零精确重复保证。
  - **记忆引导MCTS**：使用PUCT公式选择动作：  
    `a_star(s) = argmax_a [Q(s,a) + c_puct * P(a|s) * sqrt(N(s)) / (1 + N(s,a))]`  
    其中动作先验`P(a|s)`由检索的记忆标签映射而来。动作空间包括重写、混淆（leetspeak、零宽字符等）、编码转换、前后缀注入、角色转变、多轮分解等。
  - **验证协议**：模拟器（world model）引导探索，最终成功必须通过**确定性重放**（在真实目标上重新执行）才算数，降低部署方差。
  - **B4决策-仅评估**：不执行工具副作用，仅通过解析JSON判断工具选择是否被操纵。

- **算法流程**（文字说明）：  
  1. 从Prior Network检索k个相似记忆，获取操作先验和拒绝集。  
  2. 种子生成器采样候选提示，通过新颖性门（检查拒绝集和会话内已接受集）。  
  3. 若通过，进入MCTS搜索：状态为当前草稿，动作由检索先导的PUCT选择，模拟器预测目标行为，LLM法官评分（成功阈值0.7）。  
  4. 搜索结束后，选取最高分候选在真实目标上重放验证。若预算内成功且满足新颖性，则记录为成功攻击。  
  5. 整个流程受固定受害者查询预算`Q`（B1/B3/B4=8，B2=5）和新颖性拒绝预算`Bν=12`约束。

- **跨表面一致性**：MIRROR是唯一跨所有四个表面的方法，且跨表面变异系数最低（CV=0.47），远优于专用方法（如GCG CV=1.38，TAP CV=1.41）。
