## 摘要
同策略蒸馏（OPD）通过从教师模型获得密集的令牌级监督，提高了标准强化学习的学习效率。在OPD的标准KL目标中，令牌级损失被均匀平均，意味着所有令牌权重相等。然而，我们发现并非所有令牌都是平等的：随着学生生成序列变长，它们与教师分布的偏离加大，导致后面位置的监督质量下降。因此，仅使用前30%令牌的OPD可以取得与使用所有令牌相当的性能，而仅使用后30%令牌的OPD几乎学不到任何东西。本文通过约束优化的视角对该问题提供了原则性的理解。基于这些见解，我们推导出重要性加权同策略蒸馏（IW-OPD），其中每个令牌的权重取决于学生与教师分布之间的累积差异，自然地对早期令牌赋予更高权重，而对偏差较大的后期令牌赋予较低权重。我们证明，IW-OPD比OPD收敛更快，学习效率更高，并且在相同规模和跨规模设置下均达到比标准OPD更好的最终性能，在AIME-2025上性能提升高达6.9分。

## Abstract
On-Policy Distillation (OPD) improves the learning efficiency of standard reinforcement learning through dense, token-level supervision from teachers. In the standard KL objective of OPD, token-level losses are uniformly averaged, implying equal weights for all tokens. However, we discover that not all tokens are created equal: as student rollouts grow longer, they deviate further from the teacher's distribution, leading to degraded supervision quality at later positions. As a result, OPD using only the first 30% of tokens can perform comparably to using all tokens, whereas OPD using only the last 30% of tokens barely learns anything. In this work, we provide a principled understanding of this issue through the lens of constrained optimization. Based on these insights, we derive Importance-Weighted On-Policy Distillation (IW-OPD), in which the weight assigned to each token depends on the accumulated discrepancy between the student's and teacher's distributions, naturally upweighting earlier tokens and downweighting later ones with larger deviations. We show that IW-OPD converges significantly faster than OPD, with better learning efficiency, and achieves better final performance than standard OPD in both same-size and cross-scale settings, improving performance up to 6.9 points on AIME-2025.

- **背景**：同策略蒸馏（OPD）通过教师模型提供密集的令牌级监督，显著提高了标准强化学习（如 RLVR）中稀疏轨迹奖励的学习效率。OPD 的目标是最小化学生与教师分布之间的反向 KL 散度，但对所有令牌的损失进行均匀平均。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程
### 核心思想
- 将 OPD 训练视为一个 **有限预算约束优化问题**：在一个局部更新区域内（用 KL 散度半径 ρ 约束），寻找一个最优分布 q*，使其在逼近教师分布的同时，尽可能靠近当前学生分布 πθ。
- 该约束优化问题的解表明：最优学生分布 q* 是对原始学生分布 πθ 进行 **似然比（teacher/student）加权** 后的分布，权重 α 取决于有效步长预算。这解释了位置偏差——后期令牌的似然比迅速下降，因此在最优解中应被赋予更低权重。
- 基于此，作者提出 **重要性加权同策略蒸馏（IW-OPD）**，通过令牌级重要性权重调整每个令牌的梯度贡献。

### 关键技术细节
1. **约束优化视角**（Proposition 1）：最优策略 \( q^*_{\theta}(y) = \frac{\pi_{\theta}(y)}{Z_{\alpha}(\theta)} \left( \frac{\pi_T(y)}{\pi_{\theta}(y)} \right)^{\alpha} \)，其中权重依赖于前缀似然比。
2. **重要性加权目标**（Proposition 2）：通过对最优分布进行重要性采样，得到轨迹级目标，并导出令牌级代理损失：\( J^*_{\text{IW}}(\theta) = -\mathbb{E}_{y \sim \pi_{\theta}} \left[ \sum_{t=1}^T \text{sg}[\tilde{r}_t] \log \frac{\pi_{\theta}(y_t|y_{<t})}{\pi_T(y_t|y_{<t})} \right] \)，其中 \(\tilde{r}_t\) 是归一化的前缀似然比。
3. **稳定化技巧**：
   - **对数尺度缩放**：将似然比转化为对数空间：\(\tilde{r}^{\log}_t = \alpha \sum_{k < t} A^{\text{OPD}}_k\)。
   - **无符号累积**：取绝对值以避免正负抵消：\(\tilde{r}^{\text{abs}}_t = -\sum_{k < t} |A^{\text{OPD}}_k|\)。
   - **归一化**：将权重缩放到 [0,1] 区间：\(\tilde{r}^{\text{norm}}_t = 1 - \frac{\sum_{k<t} |A^{\text{OPD}}_k|}{\sum_{k<T} |A^{\text{OPD}}_k|}\)。
   - **与 OPD 插值**：最终权重为 \(\tilde{r}^{\text{IW-OPD}}_t = 1 + \gamma \cdot \tilde{r}^{\text{norm}}_t\)，默认 γ=0.5。
4. **算法流程**（Algorithm 1）：
   - 从当前学生策略采样轨迹。
   - 计算每个令牌的 OPD advantage \(A^{\text{OPD}}_t\)。
   - 计算累积不一致性，生成权重 \(\tilde{r}^{\text{IW-OPD}}_t\)。
   - 将加权 advantage \(A^{\text{IW-OPD}}_t = \text{sg}[\tilde{r}^{\text{IW-OPD}}_t] A^{\text{OPD}}_t\) 用于 PPO 更新。
