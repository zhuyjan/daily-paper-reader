## 摘要
世界模型为长视界LLM智能体提供了一种原则性的方式来赋予其预见能力：在执行前预测行动后果。然而，不可靠的预见可能被忽略、误用，甚至损害下游决策。本文提出WorldEvolver，一个自我进化的世界模型框架，它在保持下游智能体和所有模型参数冻结的同时，修订其部署时上下文。WorldEvolver整合了三个模块：（i）情景记忆，通过基于检索的模拟利用真实行动转换；（ii）语义记忆，从预测-观察不匹配中提取持久启发式规则；以及（iii）选择性预见，在将预测整合到智能体推理上下文之前过滤低置信度预测。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上测量世界模型预测准确率，在AgentBoard上测量下游智能体成功率。大量实验表明，WorldEvolver在三个基线上达到了最高预测准确率，并在下游智能体成功率上领先其他世界模型基线，展示了测试时记忆修订提升了预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

- **核心问题**：如何在不更新模型参数、不改变下游智能体策略的前提下，让世界模型在部署过程中持续修正其预测质量，并避免不可靠预测干扰智能体决策。
- **整体含义**：提出一种**训练时无需梯度更新**的自我进化世界模型框架，通过外部记忆的在线修订来提升预测保真度和下游任务成功率，为模型基强化学习中的“信任模型”问题提供语言智能体的解决方案。

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

### 2.1 核心思想

- 保持世界模型和智能体的所有参数冻结，仅通过在推理时动态修订**非参数化记忆**（external memory）来进化世界模型。
- 记忆分为两类：**情景记忆（Episodic Memory, M^E）** 存储已执行的真实转换；**语义记忆（Semantic Memory, M^S）** 存储从预测-观察不匹配中提取的持久性启发式规则。
- 增加**选择性预见（Selective Foresight）** 模块：当预测置信度低于阈值τ时，不让该预测暴露给智能体，以避免误导。

### 2.2 关键技术细节

- **情景记忆**：存储已实现的状态-动作-下一状态三元组 `(o_i, a_i, o_{i+1})`。给定候选动作 a_t，检索 k_ME 个最相似历史动作的转换（基于Jaccard相似度），作为上下文提供给世界模型。
- **语义记忆**：当预测观测 `ˆo_{t+1}` 与实际观测 `o_{t+1}` 在因子化状态空间（提取主语-谓词-宾语三元组）中出现差异时，调用LLM批评者将不匹配转化为启发式规则（例如“检查对象不会移动它”）。每条规则附带一个证据分数 e_i，初始为1，之后根据后续观测是否支持/矛盾而增减（±1/|M^S|）。仅保留 e_i > 0 的规则。
- **选择性预见**：世界模型输出预测的token序列 y_{1:n}，计算其几何平均token概率 q_t = exp( (1/n) Σ log p_θ(y_i | ...) )，若 q_t ≥ τ，则将预测暴露给智能体；否则不提供预见。
- **算法流程（Algorithm 1）**：
  1. 智能体采样一个候选动作 a_t^(0)。
  2. 从情景记忆中检索与 a_t^(0) 最匹配的 k_ME 个转换。
  3. 世界模型基于当前状态、候选动作、情景记忆和语义记忆产生预测 ˆo_{t+1}及其置信度 q_t。
  4. 选择性预见决定是否将预测传到智能体（Ft = ˆo_{t+1} 或 ∅）。
  5. 智能体基于 (s_t, F_t) 采样最终执行动作 a_t。
  6. 若 a_t ≠ a_t^(0)，世界模型重新为 a_t 做一次预测（用于语义记忆更新）。
  7. 执行动作，接收真实观测 o_{t+1}。
  8. 更新情景记忆：追加 (o_t, a_t, o_{t+1})。
  9. 若预测与观测在因子化后不一致，累积或立即生成语义规则更新 M^S。

### 2.3 公式（文字说明）

- 世界模型预测：`(ˆo_{t+1}) ~ W_θ(· | s_t, a_t, M^E_t, M^S_t)`
- 置信度：`q_t = exp( (1/n) Σ log p_θ(y_i | y<i, ...) )`
- 选择性预见输出：`F_t = { ˆo_{t+1} if q_t ≥ τ; ∅ otherwise }`

## 6. 论文的主要结论与发现

1. **WorldEvolver达到最高预测准确率**：在所有三个基座模型和两个环境下，Exact Match、Token F1、Cosine Similarity均显著优于Zero-Shot、RAWM-ϕ和ITP-I。完整版本（结合Episodic和Semantic Memory）最佳。

4. **在线持续学习有效**：随trial次数增加（图4），WorldEvolver的成功率曲线斜率更陡，说明记忆积累有助于后续重规划。
5. **置信度与准确率正相关**：高置信度预测的Exact Match更高，支持选择性预见的设计。

## 7. 优点
