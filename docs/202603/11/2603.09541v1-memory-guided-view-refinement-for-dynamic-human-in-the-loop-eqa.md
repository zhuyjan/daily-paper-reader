## 摘要
具身问答（EQA）传统上是在时间稳定的环境中进行评估的，在这些环境中可以可靠地积累视觉证据。然而，在动态且有人居住的场景中，人类活动和遮挡引入了显著的感知非平稳性：任务相关的线索是瞬态的且依赖于视角的，而“先存储后检索”的策略会过度积累冗余证据并增加推理成本。这种设置给 EQA 智能体带来了两个实际挑战：解决由视角相关的遮挡引起的歧义，以及维持精简且最新的证据以实现高效推理。为了对这一设置进行系统研究，我们引入了 DynHiL-EQA，这是一个包含两个子集的人机协同 EQA 数据集：一个具有人类活动和时间变化的动态子集，以及一个具有时间稳定观测结果的静态子集。为了应对上述挑战，我们提出了 DIVRR（动态感知视角细化与相关性引导的自适应记忆选择），这是一个无需训练的框架，它将相关性引导的视角细化与选择性记忆准入相结合。通过在提交模糊观测结果之前对其进行验证，并仅保留信息丰富的证据，DIVRR 在提高遮挡下鲁棒性的同时，通过精简的记忆保持了快速推理。在 DynHiL-EQA 和已有的 HM-EQA 数据集上进行的广泛实验表明，DIVRR 在动态和静态设置下均一致优于现有基准，同时保持了较高的推理效率。

## Abstract
Embodied Question Answering (EQA) has traditionally been evaluated in temporally stable environments where visual evidence can be accumulated reliably. However, in dynamic, human-populated scenes, human activities and occlusions introduce significant perceptual non-stationarity: task-relevant cues are transient and view-dependent, while a store-then-retrieve strategy over-accumulates redundant evidence and increases inference cost. This setting exposes two practical challenges for EQA agents: resolving ambiguity caused by viewpoint-dependent occlusions, and maintaining compact yet up-to-date evidence for efficient inference. To enable systematic study of this setting, we introduce DynHiL-EQA, a human-in-the-loop EQA dataset with two subsets: a Dynamic subset featuring human activities and temporal changes, and a Static subset with temporally stable observations. To address the above challenges, we present DIVRR (Dynamic-Informed View Refinement and Relevance-guided Adaptive Memory Selection), a training-free framework that couples relevance-guided view refinement with selective memory admission. By verifying ambiguous observations before committing them and retaining only informative evidence, DIVRR improves robustness under occlusions while preserving fast inference with compact memory. Extensive experiments on DynHiL-EQA and the established HM-EQA dataset demonstrate that DIVRR consistently improves over existing baselines in both dynamic and static settings while maintaining high inference efficiency.

*   **瞬态性与视角依赖**：人类活动和遮挡使得关键证据（如“人在做什么”）仅在特定时刻和特定视角下可见。
*   **冗余与效率矛盾**：现有的“先存储后检索”策略在长时间探索中会积累大量重复或低质量的观测，导致推理成本激增且难以处理动态变化。

### 2. 论文提出的方法论：DIVRR 框架
论文提出了 **DIVRR**（Dynamic-Informed View Refinement and Relevance-guided Adaptive Memory Selection），这是一个**无需训练（Training-free）**的框架，核心思想是“在存入记忆前先验证，只保留高价值证据”。

*   **目标区域推理（Target-Region Reasoning）**：利用视觉语言模型（VLM）计算当前观测 $O_t$ 与问题 $Q$ 的相关性得分 $s_t$。通过提取 VLM 的 Logits 并进行归一化，获得 0 到 1 之间的连续得分。
*   **相关性引导的视角细化（View Refinement）**：
    *   **触发机制**：当得分处于“模糊区间”（即 $\tau_{rot} \le s_t < \tau_{mem}$）时，触发主动感知。
    *   **多视角增强**：智能体在原地进行小幅度旋转，采集 $K$ 个辅助视角，并从中选择相关性得分最高的视角作为“验证后的观测”。
*   **自适应记忆准入（Adaptive Memory Admission）**：
    *   **准入控制**：只有当验证后的观测得分超过阈值 $\tau_{mem}$ 且通过质量过滤时，才将其存入长期记忆 $M_t$。
    *   **精简表示**：记忆中仅存储 CLIP 嵌入向量、位姿和原始图像，避免了构建复杂的 3D 语义地图或场景图带来的开销。
*   **决策与回答**：利用精简记忆引导下一步探索动作，并在探索结束后由 VLM 根据记忆生成最终答案。

### 6. 主要结论与发现
*   **动态场景优势显著**：在 DynHiL-EQA 的动态子集上，DIVRR 比最强基准提升了 **10.1%** 的准确率。

*   **静态场景泛化性**：在静态 HM-EQA 数据集上，DIVRR 依然保持领先（63.8% 准确率），且记忆量仅为 MemoryEQA 的 8% 左右。

### 7. 优点
*   **无需训练**：直接利用现有的基础模型（VLM/CLIP），易于部署和迁移。
*   **平衡了准确性与效率**：通过“按需细化”而非“全量存储”，解决了具身智能体长期运行中的记忆爆炸问题。
*   **数据集贡献**：DynHiL-EQA 填补了人机协同动态场景下 EQA 评估的空白，对后续研究具有参考价值。

### 8. 不足与局限
*   **长时序一致性**：目前主要关注单点的视角细化，对于跨越较长时间跨度的复杂动态过程（如人类连续动作的逻辑演变）缺乏显式的时间建模。
*   **社交合规性**：虽然考虑了人的存在，但智能体在探索时并未深入考虑社交规范（如避免打断人类对话或保持社交距离）。
*   **实时性挑战**：尽管比场景图方法快，但依赖大型 VLM 的在线推理仍存在秒级的延迟，在极高动态的环境中可能响应不及。
