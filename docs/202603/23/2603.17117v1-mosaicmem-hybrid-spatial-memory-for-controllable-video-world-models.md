## 摘要
视频扩散模型正从生成简短、合理的片段转向构建世界模拟器，这些模拟器必须在相机运动、重访和干预下保持一致性。然而，空间记忆仍然是一个关键瓶颈：显式 3D 结构可以提高基于重投影的一致性，但难以描绘运动物体；而隐式记忆即使在位姿正确的情况下，也往往会产生不准确的相机运动。我们提出了 Mosaic Memory (MosaicMem)，这是一种混合空间记忆，它将 patch 提升到 3D 空间以实现可靠的定位和针对性检索，同时利用模型的原生条件化（native conditioning）来保持遵循提示词的生成能力。MosaicMem 通过 patch-and-compose 接口在查询视图中合成空间对齐的 patch，在保留应持久存在的内容的同时，允许模型对演变的内容进行修复（inpaint）。结合 PRoPE 相机条件化和两种新的记忆对齐方法，实验表明，与隐式记忆相比，MosaicMem 提高了位姿遵循度，且比显式基准模型具有更强的动态建模能力。MosaicMem 进一步实现了分钟级的导航、基于记忆的场景编辑和自回归推演（autoregressive rollout）。

## Abstract
Video diffusion models are moving beyond short, plausible clips toward world simulators that must remain consistent under camera motion, revisits, and intervention. Yet spatial memory remains a key bottleneck: explicit 3D structures can improve reprojection-based consistency but struggle to depict moving objects, while implicit memory often produces inaccurate camera motion even with correct poses. We propose Mosaic Memory (MosaicMem), a hybrid spatial memory that lifts patches into 3D for reliable localization and targeted retrieval, while exploiting the model's native conditioning to preserve prompt-following generation. MosaicMem composes spatially aligned patches in the queried view via a patch-and-compose interface, preserving what should persist while allowing the model to inpaint what should evolve. With PRoPE camera conditioning and two new memory alignment methods, experiments show improved pose adherence compared to implicit memory and stronger dynamic modeling than explicit baselines. MosaicMem further enables minute-level navigation, memory-based scene editing, and autoregressive rollout.

**核心问题：** 现有的空间记忆方案存在两极分化：
*   **显式记忆（Explicit Memory）：** 依赖 3D 点云或高斯泼溅，几何一致性好但难以处理动态物体，且全局重建容易累积误差。
*   **隐式记忆（Implicit Memory）：** 将过去帧作为上下文，能处理动态内容但相机控制不精确，容易产生空间漂移，且冗余度高、计算开销大。
**整体含义：** 本文提出 **MosaicMem**，一种混合空间记忆机制，旨在结合两者的优点，实现既能精确控制相机运动，又能灵活生成动态内容的长时一致视频生成。

### 2. 方法论：核心思想与关键技术
**核心思想：** 采用“图像块（Patch）”作为记忆的基本单元。将 Patch 提升到 3D 空间进行定位，在生成新视角时，像拼贴“马赛克”一样将相关的记忆块检索并对齐到当前视图中。

**关键技术细节：**
*   **Mosaic Memory 检索：** 利用离线 3D 估计器获取深度和位姿，将历史 Patch 投影到当前查询视角。模型只需填充（Inpaint）未见区域或更新动态区域。
*   **记忆对齐机制（Warping）：**
    *   **Warped RoPE：** 根据 3D 几何关系重新计算旋转位置编码，使记忆块在潜空间中实现像素级的坐标对齐。
    *   **Warped Latent：** 在特征空间对检索到的 Patch 进行双线性插值采样，进一步修正 VAE 压缩带来的对齐误差。
*   **PRoPE 相机控制：** 引入投影位置编码（Projective Positional Encoding），将相机投影矩阵直接注入自注意力机制，增强模型对复杂轨迹的遵循能力。
*   **Mosaic Forcing：** 结合 Causal Forcing 等技术，将双向扩散模型转化为高效的自回归生成器，支持实时交互。
